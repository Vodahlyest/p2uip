💬 p2uip — Peer-to-You-IP Messenger

p2uip — это легковесный кроссплатформенный P2P-мессенджер, работающий прямо в терминале (TUI).

Минималистичный, быстрый и без центральных серверов — только прямое соединение между пользователями.

📌 О проекте

Этот проект разработан как практическая платформа для изучения языка Rust и включает:

асинхронное программирование
работу с TCP-сокетами
архитектуру P2P-приложений
построение терминальных интерфейсов (TUI)

🚀 Установка
📦 Готовые сборки (рекомендуется)

Перейдите в раздел Releases и скачайте архив для вашей ОС.

🐧 Linux
tar -xzf p2uip_linux.tar.gz
cd p2uip
sudo ./install.sh

После установки:

p2uip

Требуются root-права для настройки /opt и firewall.

🪟 Windows
Распакуйте p2uip_windows.zip
Запустите install.bat от имени администратора

Скрипт:

установит приложение
добавит правило в брандмауэр
создаст ярлык


🛠 Сборка из исходников
Требования
Rust
Cargo

git clone https://github.com/ВАШ_НИК/p2uip.git
cd p2uip

Исполняемый файл появится в:

⚠️ Не забудьте открыть порт 8080/tcp для режима сервера.


🎮 Использование
Запустите:
p2uip
Выберите режим:
1 — Client
2 — Server
Введите никнейм
🖧 Подключение

Если вы Host:

получите локальный IP (например 192.168.1.15:8080)
передайте его собеседнику

Если вы Client:

введите IP адрес сервера
💬 Управление
Действие	Клавиша
Ввод сообщения	i
Отправка	Enter
Выход из ввода	Esc
🧱 Архитектура и зависимости

Проект использует ключевые библиотеки экосистемы Rust:

tokio — асинхронный runtime
ratatui — TUI интерфейс
crossterm — обработка ввода/вывода
serde / serde_json — сериализация сообщений
local-ip-address — определение IP
